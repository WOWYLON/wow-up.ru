== Description ==

Declares a plugin that will show contact information.


== Installation ==

To install this plugin:

1. Upload the `contact-info-widget` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it!

Alternatively you can search for the plugin from your WordPress dashboard and install from there.
