== Description ==
A blank child theme for Vixan.

For more information about Vixan please go to http://themeforest.net/user/3jon

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button and click 'Upload Theme'.
2. Extract .zip file and upload 'vixan-child.zip' file
3. Click on the 'Activate' button to use your new theme right away.
4. Go to https://3jon.com/documentation/wordpress/vixan/ for a guide on how to customize this theme.
5. Navigate to 'Vixan Options' in your admin panel and customize to taste.
