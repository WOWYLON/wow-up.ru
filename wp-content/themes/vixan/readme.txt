== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button and click 'Upload Theme'.
2. Extract .zip file and upload 'vixan.zip' file
3. Click on the 'Activate' button to use your new theme right away.
4. Go to https://thememarch.com/documentation/wp/vixan/ for a guide on how to customize this theme.
5. Navigate to 'Vixan Options' in your admin panel and customize to taste.
6. For getting better result please follow the given documentation.
